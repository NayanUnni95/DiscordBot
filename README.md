# Discord Bot

### Create new python Environment

```bash
python3 -m venv venv
```

### Activate Env

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Update after install dependencies

```bash
pip freeze > requirements.txt
```

### Run Python Script

```bash
python3 main.py
```

- Interact with users
